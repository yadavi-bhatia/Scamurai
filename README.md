🚨 Scamurai

Real-time AI Scam Call Detection & Prevention

🎯 Overview

Scamurai analyzes phone call audio in real time to detect scams using multiple signals like language, impersonation, financial intent, and voice patterns. It generates a live risk score and alerts the user during the call.

⚡ Features
Real-time audio → transcript → risk analysis
Multi-signal detection (linguistic, impersonation, money, voice)
Live risk score dashboard
Alerts for suspicious activity
Evidence logging (transcript + signals)
